02 — Why it works

It's 1/7 in disguise.

The decimal expansion of 1/7 is a repeating block of six digits. That block is 142857.

Long division of 1 ÷ 7

Watch the remainders as we divide. Each remainder is the next numerator, and each remainder corresponds to a different sevenths fraction.

The remainders cycle through 1 → 3 → 2 → 6 → 4 → 5 → 1 → … — every nonzero number mod 7 appears exactly once before repeating. That's why 7 is a full-reptend prime.

Every seventh is a rotation

Because the remainders come back around, each k/7 has the same six digits — just starting at a different point in the cycle.

The one-line proof

Since 1/7 = 0.142857…, multiplying by 106 shifts the decimal by six places:

106 / 7  =  142857.142857…
106 / 7  −  1 / 7  =  142857
(106 − 1) / 7  =  999999 / 7  =  142857

So 142857 × 7 = 999999. Multiplying 142857 by k (for k = 1…6) is the same as taking k × (10⁶ − 1)/7, which is why the result is always a cyclic permutation.

04 — Bonus

Is it divisible by 7?

A companion trick using mod 7. Chop off the last digit, double it, and subtract. Repeat. If the result is divisible by 7, so was the original.

Why does this work?

If N = 10a + b (where b is the last digit), then N ≡ 0 (mod 7) iff 10a + b ≡ 0 (mod 7). Multiplying by 5 (which is invertible mod 7): 50a + 5b ≡ 0, i.e. a + 5b ≡ 0 (mod 7). Subtract 7b from both sides: a − 2b ≡ 0 (mod 7). So N is divisible by 7 iff a − 2b is.
